# VVOX

> **One Timeline. Multiple Audio Worlds.**

VVOX is a modern multi-channel media playback platform designed for synchronized video presentation with independently routed audio outputs.

Whether you're delivering multilingual presentations, live interpretation, theatrical productions, museums, educational content, or worship services, VVOX enables a single video timeline to drive multiple synchronized audio channels simultaneously.

---

## Vision

Traditional media players assume one video and one audio output.

VVOX is built around a different idea:

> One visual experience can serve many audiences at the same time.

A single presentation can deliver multiple languages, commentary tracks, assistive listening, monitoring feeds, or custom mixes—each routed independently to dedicated output devices.

---

## Key Features

* Multiple synchronized audio outputs
* Independent physical channel routing
* Frame-accurate audio synchronization
* Professional ASIO support
* Preset-based workflow
* Modern desktop UI
* Cross-platform architecture
* Extensible plugin-ready design

## Current Development Status

VVOX is currently under active development.

The initial milestone focuses on building the desktop application foundation:

* Avalonia UI
* MVVM architecture
* Preset management
* Design-time mock data
* Runtime mock playback
* UI infrastructure

Future milestones will introduce:

* Real video playback
* ASIO audio engine
* Multi-device routing
* Timeline synchronization
* Plugin architecture
* Advanced media control
